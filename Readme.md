This project is a full-stack e-commerce application built using the MERN stack.  
The goal of this project is not UI replication, but to demonstrate **real-world backend architecture, data modeling, authentication flows, and scalable system design**.

## Tech Stack
- **Frontend:** React (planned)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** Passwordless authentication (OTP + OAuth ready)
- **Deployment:** Planned (Vercel / Render / MongoDB Atlas)

## Core Features (Planned)
- User authentication (OTP / OAuth)
- Product browsing and categorization
- Product details page
- Cart and checkout flow
- Order creation and order history
- Admin product management

## Project Focus
This project focuses on:
- Proper MongoDB data modeling
- Secure authentication without passwords
- Clean API design
- Scalable schema relationships
- Production-grade backend practices
